# Library Management System

This is a simple Library Management System built using C++.
It allows basic operations like adding books, managing members, issuing and returning books, and searching records.

## Features

* Add new books and members
* Issue and return books
* Search books by title or author
* Display available books
* Stores data using file handling

## Tech Used

* C++
* File Handling
* Basic HTML, CSS (for viewing data)

## How to Run

1. Compile the program:
   g++ main.cpp Library.cpp FileHandler.cpp -o library.exe

2. Run:
 ./library.exe

3. Open index.html to view stored data in browser.
